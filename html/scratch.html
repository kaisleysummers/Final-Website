Steps to Learn How to Embroider

  1. Buy Materials
    • Embroidery Hoop
    • Fabric

  2. Learn Basic Stitches
    • Backstitch
    • Satin
    • French Knot
    • Lazy Daisy
    • (See above vidio for these and others)
  3. Practice
    • Do each stitch until it looks right
    • Follow a pattern
  4. Start Your First Project
    • Look up inspiration
    • Sketch out what you want
    • Start Embroidering!
